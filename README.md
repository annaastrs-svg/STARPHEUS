#STARPHEUS BOT

(foto proyecto completo)

Orpheus Bot It's a semi-autonomous robot inspired by the Hack Club mascot (Orpheus).
Starpheus is a Wall-E type robot.
You can control it with a LogiTECH controller (you control its neck with an MG996R servo motor and the TT motors). 
It has 3 OLEDs that make up its eyes and mouth, simulates emotions with basic state machines (angry, happy, sleepy,
neutral), has an infrared sensor on its head that detects 'pets,' 
and uses an HC ultrasonic sensor to avoid collisions :B

NOTE: STILL IN CONSTRUCTION! ill Computer vision for face detection and I will make a better chasis.

The design was made so that our little Starpheus isn't heavy and has good mobility.
It has holes for OLED screens, a reset button on the back, the ESP32-S3 connector, the infrared sensor, the ultrasonic sensor, and spots to attach the motors.

I'm an amateur, this is my biggest solo project so far, and I like to challenge myself.
I prefer making mistakes to asking Gemini, so I thought of something that would be cute, interactive, and mobile.
I don't remember exactly why I thought of Orpheus lol. 

I was somehow inspired by the aesthetic of ABB robots, I don't know why I feel like they have emotions...
